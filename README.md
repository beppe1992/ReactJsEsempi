# ReactJsEsempi
Questo progetto racchiude alcuni esempi di applicazioni sviluppate con ReactJs.

## HelloWorldExample
Questo progetto contiene un semplice Hello World fatto con ReactJs <br>

### Framework
Questo progetto utilizza ReactJs. Non è presente alcun servizio REST.

### Configurazioni
Per provare il progetto e' necessario aver installato nmp, e bower.
Una volta scaricato il progetto è necessario lanciare i comandi "bower init" e "bower update" per inizializzare le dipendenze Javascript utilizzate dal progetto.

### Link
Esempio HelloWorld: http://localhost:8080/Index.html <br>
